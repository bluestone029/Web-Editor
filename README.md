# Tempo

> 项目基于angular2 (版本号 7.0.2)开发,使用ng-cli构建完成，主要实现:

* 拖拽创建组件
* 页面管理
* 元素列表
* 样式绑定
* 快捷键
* 预览

## 下载

```
git clone https://github.com/bojue/H5-Editor.git
cd H5-Editor
npm install
```

## 启动

>  npm run start

默认端口4200
