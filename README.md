## 下载

```javascript
git clone https://github.com/bojue/H5-Editor.git
cd H5-Editor
npm i
npm run start
```

在浏览器打开：`http://localhost:4200`,端口默认使用4200

### 文档

> 功能列表

* 拖拽创建组件
* 页面管理
* 元素列表
* 样式绑定
* 快捷键
* 预览

> 快捷键

* 拷贝 `control+ c`
* 复制 `control+ v`
* 删除 `delete`
* 上下左右快捷键
